## Задание 1 (`task1.cpp`)

### Описание
- Создание массива из N элементов, заполнение случайными числами от 1 до 100.  
- Нахождение максимального и минимального значения массива.  
- Вывод массива и найденных значений.  
- Параллельная обработка с использованием OpenMP (`#pragma omp parallel for`).  
- Сравнение времени выполнения последовательной и параллельной реализации.

### Цель
- Ознакомление с базовыми операциями над массивами.  
- Практика измерения времени выполнения и ускорения с использованием OpenMP.  

### Файл решения
- `task1.cpp` содержит полный код задания с последовательной и параллельной реализацией.

## Задание 2 (`task2.cpp`)

### Описание
- Реализация динамических структур данных:  
  - Односвязный список с операциями `add`, `remove`, `search`.  
  - Стек с операциями `push`, `pop`, `isEmpty`.  
  - Очередь с операциями `enqueue`, `dequeue`, `isEmpty`.  
- Параллельная работа с структурами данных с использованием OpenMP: одновременное добавление элементов.  
- Сравнение производительности последовательного и параллельного выполнения.

### Цель
- Изучение работы со структурами данных и их методов.  
- Практика параллельного выполнения операций с динамическими структурами.  

### Файл решения
